I'm inviting anyone who's interested to help improve the world's stock of digital Sanskrit texts by using skrutable's en-masse meter-identification functionality to help sniff out and fix textual problems and to establish a meter dataset for reference and benchmarking purposes.

how-to

Analyzing the meter of a whole metrical text is a fun and effective way to identify textual problems in need of fixing. Obviously you need to already know a bit about Sanskrit and Sanskrit meter, and you have to start with decently good data. But assuming those, then it's only a matter of having the right tools and patiently applying yourself. Here's one way to go about it.

Step 1: Identify a text you'd like to work on. Go find it online. GRETIL texts include some of the highest-quality ones around and are relatively easy to work with, so they're a good starting point. For example: The Bhagavadgītā (BhG) (raw GRETIL file)

Step 2: Clean up the file so that it is a text file with exactly one (potential) whole verse per line. Half of an anuṣṭubh śloka is ok, too. For example: BhG input cleaned

Step 3: Submit the cleaned file to skrutable's en-masse meter identifier using the "whole file" button. For example: BhG raw output >> BhG output cleaned

Step 4: Read through the results, looking for irregularities (use regexes to search more effectively). Where there's a metrical problem, there's usually a textual problem. For example: (BhG doesn't have any, but Rāmāyaṇa does...)

Step 5: If you can, try to come up with solutions for the textual problems, one-by-one.

  • Is the problem in the digital text itself?
  • Or in the edition the digital text is based on?
  • Is it skrutable?
  • Something even more interesting?
Once you're confident about your results, send the results over to me. I'll post the cumulative results here. This takes time, patience, and expert knowledge, so there's no way one person could ever do it properly all on their own!

And if you were able to make improvements to the text, send it on back to GRETIL. I'm happy to help you do that, if need be.
